# cricket_mania

Cricket Mania is a cricket live score Application. Get Live Cricket Scores, Scorecard, Schedules of International and Domestic cricket matches.

### How to run


###### Prerequisite
- Flutter 3.7.3 (Stable)
- Dart 2.19.2

###### Build
```
$ flutter build apk --release
```

###### Run
```
$ flutter run 
```
